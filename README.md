# Comp-104

Bu depo, Comp-104 dersi kapsamında C diliyle yazılmış örnekler ve projeler içerir.

## Klasörler
- c-kodlari: C diliyle yazılmış kodlar

## Başlangıç

1. Depoyu klonlayın:
	```sh
	git clone <repo-url>
	```
2. Kendi C dosyalarınızı `c-kodlari` klasörüne ekleyin.
3. Derlemek için örnek:
	```sh
	gcc c-kodlari/ornek1.c -o c-kodlari/ornek1.out
	```
